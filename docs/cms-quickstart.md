# Adding an Event — Quick Start
### The Wandering Lantern CMS

**→ thewanderinglantern.com/cms/**
Log in with your GitHub account.

---

## The Short Version

90% of events are free with a signup form. Here's all you need:

1. Click **Events** → **New Event**
2. Fill in these 7 fields:

| Field | What to type |
|---|---|
| **Title** | The event name |
| **Date** | Pick from the calendar |
| **Time** | e.g. `6:00 PM - 7:00 PM` or `11:00 AM` |
| **Age Range** | Usually `All ages` or something like `6-12` |
| **Category** | Pick one (see below) |
| **Excerpt** | 1-2 sentences — the most interesting thing about this event |
| **Mailchimp Form ID** | Always `f1915b185f` — don't change this |
| **Mailchimp Tag ID** | Usually `3728680` — see tag list below |

3. Click **Publish** → done. Live in ~2 minutes.

**Not sure about something? Save as Draft and ask Michael.**

## Category — Pick the Right Badge

| Pick this | When... | Looks like on the site |
|---|---|---|
| **Author Visit** | A visiting author or special guest | Gold filled badge — most prominent |
| **Workshop** | Craft, art, skill-based programs | Brown outlined badge |
| **Special Event** | Seasonal or one-of-a-kind | Red outlined badge |
| **Story Time** | A one-off storytime session | Gold outlined badge |

---

## Mailchimp Tag ID — Pick from the Built-in Picker

Below the Mailchimp Tag ID field you'll see a panel: **"Browse all tags or create a new one"**. Click it to expand.

- **Picking an existing tag:** use the **Existing** dropdown — it lists every tag in Mailchimp by name. Selecting one fills in the ID automatically.
- **Creating a new tag:** type a name in the **Or create** field and click **Create**. The tag is created in Mailchimp instantly and the ID auto-fills.

**Common choices:**

| Tag name | Use for |
|---|---|
| General Events (3728680) | Most one-off events |
| Family Classics Book Club (3728683) | Every month's book club |
| Saturday Story Time (3728684) | Saturday storytime |

For brand-new recurring series, just type a clear name and click Create — no need to leave the CMS or ask Michael.

## Is Your Event Different?

**Paid event (tickets through the store)?**
Fill in **Ticket Purchase URL** with the Shopify or Eventbrite link. Leave Mailchimp fields blank.

**Drop-in / no signup needed?**
Leave ALL registration fields blank. Just fill in the basics.

**Author visit with a special guest?**
Also fill in: **Special Guest Name**, **Guest Role**, **Featured Book**.
This adds a gold ★ to the Saturday Story Time banner automatically if the visit is on a Saturday.

**Multi-day camp?**
Set Date to the first day. Fill in **Date Range** (e.g. `Monday–Friday, June 8–12, 2026`).

**Have a flyer image?**
Paste the image URL into **Flyer Image URL**. It shows up automatically.

---

## Marking an Event Full or Cancelled

Change **Status** to **Full** or **Cancelled**. That's it — the registration form is replaced automatically with the right message. You don't need to edit anything else.

---

## When Things Go Wrong

**Not sure if it looks right?** Save as Draft first. Visit the event URL directly to preview it before publishing.

**Made a mistake after publishing?** Just edit it and re-publish. Changes go live within 2 minutes.

**Something broken or confusing?** Ask Michael. Don't guess.
