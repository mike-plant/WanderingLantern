Three Ways to Use This Dashboard

Option 1: Digital Dashboard (Easiest)

  • Open index.html in your browser daily
  • Bookmark it for quick access
  • Check today's task and click through sections as needed
  • Progress saves to your browser automatically

Pros: No printing, always accessible, works on phone/tablet/laptop
Cons: Progress is device-specific unless you use Google Sheets option

Option 2: Print-Friendly (Classic)

  • Print week-by-week.html (one week at a time)
  • Put it on your fridge with a magnet
  • Check off tasks with a pen
  • Print next week when done

Pros: Visible daily reminder, satisfying to check off tasks, low-tech
Cons: Uses paper, need to print each week

Option 3: Google Sheets (Cross-Device)

  • Follow setup guide in SETUP-GOOGLE-SHEETS.md
  • Open poc-sheets.html on all your devices
  • Tasks sync automatically every 30 seconds
  • Emily, Michael, you - all see the same data

Pros: True cross-device sync, real-time collaboration, see progress from anywhere
Cons: 10-15 min one-time setup required

üí° Recommendation: Start with Option 1 (digital dashboard) TODAY. Try the Google Sheets option this weekend if you want cross-device sync. Keep a printed week on your fridge as a backup visual reminder.

Frequently Asked Questions

How do I save my progress?

If using the digital dashboard, progress saves automatically to your browser. If using Google Sheets option, it syncs across all devices. If printing, mark with a pen!

Can I use this on my phone?

Yes! All pages are mobile-friendly. Bookmark the dashboard on your phone's home screen for quick access.

What if I skip a task?

That's totally fine! Open the Modular Plan to see what you can safely skip. Most things are optional.

Do I need the Google Sheets integration?

No! It's optional. Use it if you want cross-device sync. The dashboard works great without it.

Can I share this with Emily and Michael?

Absolutely! Email them the launch-dashboard folder, or use the Google Sheets option to share tasks.

What if I want to print everything?

Each page has print styles. Press Cmd+P (Mac) or Ctrl+P (Windows) on any page to print or save as PDF.

How often should I check the dashboard?

Daily for 2 minutes to see today's task. Weekly on Sundays/Fridays for planning. That's it!
